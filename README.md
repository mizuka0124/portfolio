# Portfolio

### Yubisuma
androidstudioでJavaを使用して作った指スマゲーム

参加人数を決め、自分が親の時は合計数を予測し手を出す。

両手の親指を立てるか立てないかで0か1か2を自分の手として出す

親の予測数がプレイヤー全員の手の合計と合致した場合親の片手を引く

両方の手がなくなった人の勝ち

一人が勝てばゲームは終わり

### ARFoundation
unityのARFoundationでC#を使用して作ったARアプリ

平面を検知し検知した部分をタップすると選択したオブジェクトを生成することができる

生成したオブジェクトをタップすると選択状態にできる。

指を動かすことで移動

向き変更ボタンを押すことでy軸方向への回転

色ボタンを押すことで色変更

オブジェクトボタンを押すことでオブジェクトの形を変更

ピンチインピンチアウトで拡大縮小


線描がモードで空間に線をかける
